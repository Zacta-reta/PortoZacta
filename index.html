Kecakapan

HTML

CSS

Memahami CSS untuk mendesain dan memodifikasi tampilan halaman web, mencakup pengaturan layout, warna, tipografi, dan elemen-elemen visual lainnya. Dapat membuat desain yang responsif agar tampilan website tetap optimal di berbagai perangkat, serta mengaplikasikan berbagai teknik styling dasar.
